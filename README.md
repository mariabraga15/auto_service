# Instrucțiuni de rulare a aplicatiei

```bash
# 1. Clonare repository
mkdir project
git clone https://github.com/mariabraga15/auto_service.git
cd auto_service

# 2. Instalare dependencies
npm install

# 3. Populare baza de date
npm run seed:orm

# 4. Pornire server

npm start #sau npm run dev'


# Aplicația va fi disponibilă la: http://localhost:8080/
